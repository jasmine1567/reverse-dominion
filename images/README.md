# カードイラスト置き場（SSR・UR専用）

このフォルダに各カードのイラストを置くと、ゲーム内のカード絵柄が自動で差し替わります。
- ファイル形式：**.webp**（推奨。軽量・高画質）
- ファイル名：**カードID.webp**（例：`ssr001.webp`）
- 推奨サイズ：**512×512px 前後の正方形**（カード枠内に object-fit: cover で表示）
- 透過は不要（背景込みの一枚絵でOK）

ファイルが無い場合は、これまで通り絵文字が表示されます（壊れません）。
必要なファイル名と各カードの内容は、リポジトリ直下の `ART_GUIDE.md` を参照してください。
